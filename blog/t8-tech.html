A Regular Expression is a way to describe patterns in text that can be matched and compared against. It is essentially a search query, or "a string that describes a pattern for matching elements in other strings."

Forward slashes are used to start and end a regular expression.
You can add modifiers to your regular expressions, following a character, to modify what it is your regular expression is searching for.

x = "The car costs 20000 and the cat costs $45."
x.scan(/\d/) { |x| puts x }
