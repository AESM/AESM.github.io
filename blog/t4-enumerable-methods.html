Map Map Mappin' on E-Num-Er-A-Blllles!

2014/10/27

The map method iterates though every element of an array, runs the code block, and returns a new array. If you want to modify the original array, use the map! method. The collect method is equivalent.


Example: array_1 = [2, 4, 6, 8] array_1.map { |item| p item ** 2 } 4 16 36 64


The map method will yield each member of an array to the block. But unlike each (which just returns the original array), map will collect the results in a new array and return that instead.
