#all?

The all? method iterates over each element of the array, comparing them all based on the data entered into the code block. The return value is true if all of the elements meet the criteria, or false if at least one element does not.

array_num = [1, 2, 3, 5, 10]
array_num.all? { |elem| p elem > 3 }
false

The all? method iterates over each element of the hash, comparing them all based on the data entered into the code block. The return value is true if all of the elements meet the criteria, or false if at least one element does not.
The block takes two arguments – the key & the value.

meal = { "App" => "cheesebread", "Entree" => "steak", "Dessert" => "bread" }
puts meal.all? { |key, value| value.match('bread') }
false
